# DruxtBlocksBlockMixin

Provides Vue.js properties to render Drupal Block components.

This Mixin is intended for use by block type Component Suggestions for targetted theming of Drupal Blocks.

Kind: global mixin
See: DruxtEntityComponentSuggestionMixin
Example

<script>
// Import mixin.
import { DruxtBlocksBlockMixin } from 'druxt-blocks'

export default {
  // Register mixin.
  mixins: [DruxtBlocksBlockMixin],
}
</script>
