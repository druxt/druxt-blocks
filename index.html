DruxtJS Blocks

Provides Drupal blocks and region components to be used within a Druxt (DRUpal nuXT) project.

Get started

The DruxtJS Blocks module adds Drupal (opens new window) Blocks, Content Blocks and Block Region Vue.js components to your NuxtJS (opens new window) frontend.

# The DruxtBlock component

Druxt Blocks provides a Vue.js component to render a specified Block.

<DruxtBlock :uuid="name" :mode="displayMode" />
1

Get started with the Guide and API Documentation.

# The DruxtBlockRegion component

Druxt Blocks provides a Vue.js component to render all Blocks in a specified Drupal Region, for a specified theme.

<DruxtBlockRegion :name="name" :theme="theme" />
1

Get started with the Guide and API Documentation.

# Druxt component

The DruxtBlock and DruxtBlockRegion components are compatible with the DruxtJS component theming system:

<Druxt module="block" :props-data="{ uuid, mode }" :wrapper="wrapper" />
1
<Druxt module="block-region" :props-data="{ name, theme }" :wrapper="wrapper" />
1

See the Druxt component documentation (opens new window) for more information.

# DruxtJS

DruxtJS is a suite of modules to connect a Drupal (opens new window) JSON:API backend to a NuxtJS (opens new window) frontend.

Find out more at http://druxtjs.org (opens new window)
