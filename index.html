DruxtJS Blocks

Provides Drupal blocks and region components to be used within a Druxt (DRUpal nuXT) project.

Get started

The DruxtBlocks module adds Drupal (opens new window) Blocks, Content Blocks and Block Region Vue.js components to your Nuxt.js (opens new window) application.

# DruxtBlock

The DruxtBlock component renders a Drupal JSON:API Block resource by ID or UUID.

<DruxtBlock :id="drupal_internal__id" />
1
<DruxtBlock :uuid="uuid" />
1

Get started with the Guide and API Documentation.

# DruxtBlockRegion

The DruxtBlockRegion component renders all visible blocks for the specified theme region.

<DruxtBlockRegion :name="name" :theme="theme" />
1

Get started with the Guide and API Documentation.

# DruxtJS

DruxtJS is a framework to connect a Drupal (opens new window) JSON:API backend to a Nuxt.js (opens new window) application.

Find out more at https://druxtjs.org (opens new window)
