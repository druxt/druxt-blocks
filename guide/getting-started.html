# Getting started

# Druxt.js

Druxt.js is a suite of modules to connect a Drupal JSON:API backend to a Nuxt.js frontend.

This module is included in the core Druxt.js project, the below instructions are only required if installing this module standalone.

# Quickstart

  1. Install the NPM module:

    npm i druxt-blocks
    1

  2. Add the module and configure as required in nuxt.config.js:

    module.exports = {
  modules: [
    'druxt-blocks'
  ],
  druxt: {
    baseUrl: 'https://example.com'
  }
}
